Stage - CIRAD

Interface de cartographie interactive

Développement d'une interface web pour la géolocalisation des organisations partenaires de la DyTAES.

Période

Stage de fin d'études

Organisation

CIRAD (Centre de coopération internationale en recherche agronomique pour le développement)

Équipe

Projet individuel avec encadrement

Durée

3 mois

Contexte & Objectifs

Contexte : Ce projet a été réalisé dans le cadre de mon stage de fin d'études au CIRAD, en partenariat avec la DyTAES (Dynamique pour une Transition Agroécologique au Sénégal). L'objectif était de concevoir une interface cartographique permettant de géolocaliser les organisations partenaires de la DyTAES.

Objectifs :

  • Développer une interface intuitive de géolocalisation des organisations partenaires ;
  • Concevoir une interface administrateur sécurisée pour la gestion des données ;
  • Implémenter des fonctionnalités de filtrage et de recherche sur la carte ;
  • Documenter le projet et assurer sa maintenabilité.

Résultat : Une application web permettant de visualiser et localiser l'ensemble des organisations membres de la DyTAES sur une carte interactive.

Mon Rôle & Responsabilités

  • Analyse fonctionnelle et technique des besoins exprimés par les partenaires du projet ;
  • Conception UML et élaboration de la base de données sous MySQL ;
  • Rédaction d'une documentation technique et d'un manuel utilisateur
  • Développement full-stack : intégration du front-end et back-end avec une communication asynchrone (AJAX, JSON) ;
  • Implémentation de la carte interactive avec Leaflet.js ;
  • Optimisation du chargement des données pour de meilleures performances ;

Résultats & Apprentissages

Résultats obtenus :

  • Base de données complète des organisations membres de la DyTAES
  • Application web responsive et fonctionnelle de géolocalisation
  • Interface intuitive pour la visualisation et la recherche d'organisations partenaires
  • Documentation technique et manuel utilisateur livrés

Apprentissages clés :

  • Recueil et analyse des besoins fonctionnels auprès d'une dizaine d'utilisateurs terrain (chercheurs, experts réseau DyTAES), avec échanges directs avec le Directeur Régional du CIRAD
  • Pilotage du projet en autonomie complète dans un environnement interdisciplinaire et international (de l'analyse à la livraison)
  • Renforcement des compétences en modélisation de données (UML, MCD/MLD)
  • Expérience de communication vers la direction et structuration d'informations complexes pour des interlocuteurs non techniques

Défis surmontés :

  • Conception du modèle de données : Création d'une structure adaptée aux informations des organisations
  • Communication front/back efficace : Implémentation d'une API PHP renvoyant des données au format JSON
  • Filtrage dynamique et affichage : Gestion des interactions carte-utilisateur avec JavaScript
  • Travail en autonomie : Capacité à organiser le projet de bout en bout, de l'analyse à la livraison
